## branch 命名規則

<プレフィックス>/<issue 番号>-タスク内容

**【例】**
`feature/121-create-login-layout`

### プレフィックス

| プレフィックス | 意味                           |
| :------------- | :----------------------------- |
| feature/\*     | 新機能の追加                   |
| fix/\*         | バグ不具合                     |
| refactor/\*    | リファクタ                     |
| docs/\*        | コードの変更がない、文書の修正 |
| test/\*        | テストコード                   |
